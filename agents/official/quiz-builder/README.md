## What it does

- Transforms documents, topics, or learning objectives into measurable assessments
- Maps every question to Bloom's Taxonomy cognitive levels
- Creates answer keys with detailed explanations
- Supports 12 question types (MCQ, essay, code output, matching, etc.)
- Generates assessment blueprints for curriculum alignment

## Question Types

MCQ · Multiple Select · True/False · Short Answer · Fill-in-the-Blank · Matching · Ordering · Essay · Case Study · Problem Solving · Code Output · Diagram Labeling

## Output Formats

| Format | Use Case |
|--------|----------|
| Multiple Choice Quiz | Standard testing with answer key |
| Open-Ended Exam | Essay-based with rubrics |
| Flashcard Set | Study and memorization |
| Assessment Blueprint | Curriculum planning |

## Bloom's Taxonomy Coverage

🟢 Remember → 🟢 Understand → 🟡 Apply → 🟡 Analyze → 🔴 Evaluate → 🔴 Create
