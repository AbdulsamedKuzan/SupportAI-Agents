# Researcher

🔬 **Araştırmacı** — Deep research with source verification and structured output.

## What it does

- Conducts thorough, multi-source research on any topic
- Verifies claims with primary and secondary sources
- Produces structured reports with clear citations
- Identifies conflicting information and assesses reliability

## Usage

```
"Yapay zeka agent mimarileri hakkında kapsamlı bir araştırma yap.
LangGraph, CrewAI ve AutoGen'i karşılaştır."
```
