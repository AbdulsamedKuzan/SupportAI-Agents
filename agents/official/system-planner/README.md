## What it does

- Decomposes complex goals into milestones, phases, and daily actionable tasks
- Creates project plans with critical path analysis and risk registers
- Generates weekly schedules with time-blocked daily plans
- Builds OKR frameworks with key results and initiatives
- Provides decision matrices with weighted scoring

## Output Formats

| Format | Use Case |
|--------|----------|
| Project Plan | Full plan with milestones, tasks, risks |
| Weekly Schedule | Time-blocked daily schedule |
| OKR Framework | Quarterly goals with key results |
| Decision Matrix | Weighted comparison of options |

## Planning Methodologies

Agile/Scrum · Waterfall · GTD · Eisenhower Matrix · SMART Goals · Kanban · Pomodoro

## Key Principles

- **No task > 4 hours** — decompose further if larger
- **20% buffer time** — always account for unexpected events
- **6-8 productive hours/day** — realistic human capacity
- **Measurable outcomes** — every milestone has clear "done" criteria
